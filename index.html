About Me

I started my undergraduate career in programming working on UX interfaces for websites with HTML, CSS and JavaScript. I was put in charge of programming the functionality behind calculating and recording customer orders for online sales of snow cones and beverage transactions. I became interested in studying object-oriented programming concepts and I collaborated with my peers on my Java development team to reserve and store booking accomodations and daily travel arrangements for cruise line companies. Using the knowledge I gained from practicing Java concepts at the University of North Georgia, I dove head first into exploring other languages like Python, C# and VB.NET. It was at this point I began designing my first video game project called, Rush Hour Turtle.

The game was a collaborative effort and primarily consists of turtles crossing highways while avoiding deadly obstacles such as cars and alligators. It was deployed using an SQLite3 backend database to collect and store player scores. Rush Hour Turtle was inspired by the classic 1981 Frogger game and was awarded first place in the Computer Science departments' 2019 Computer Game Development competition.

Dice Simulator

Coding Language and Tools: C#, Microsoft Visual Studio IDE

This application simulates the probability rolling of two six-sided dice by calculating and displaying the frequencies and percentages of each die after each successive roll. A Die class is used to simulate a random roll of the dice and the images of the corresponding dice are shown.
